# Student Management System

## Project Description

The Student Management System is a simple Java application used to manage student records.
It allows users to add, view, and search student details. This project demonstrates basic Object-Oriented Programming concepts in Java.

## Features

* Add new student records
* View all students
* Search students by ID
* Store student details such as ID, name, and marks

## Technologies Used

* Java
* Object-Oriented Programming (OOP)

## Project Structure

Student-management-system
├── Student.java
├── StudentManager.java
├── Main.java
└── README.md

## How to Run the Project

1. Compile the program

```
javac Main.java
```

2. Run the program

```
java Main
```
